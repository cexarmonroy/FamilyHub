# FamilyHub

App web centralizada para gestionar información familiar (perfiles, salud, colegio, agenda semanal) con **Next.js** + **Supabase**.

## Requisitos
- Node.js + npm

## Setup
1. Copia variables de entorno:
   - Duplica `.env.example` como `.env.local` y completa:
     - `NEXT_PUBLIC_SUPABASE_URL`
     - `NEXT_PUBLIC_SUPABASE_ANON_KEY`

2. Instala dependencias:
```bash
npm install
```

3. Ejecuta en desarrollo:
```bash
npm run dev
```
